# cesium

#### 介绍
可以显示香港特别行政区的部分中西区、湾仔区、东区的三维建筑模型、俯瞰图，能添加飞机模型，能够实现下雪下雨烟花特效并能停止该特效，VR切换、全屏效果，实时经纬度坐标及视角高显示，底图切换，搜索，二三维一体化，绘制并跟踪路线，测量长度以及面积的功能。 本次课设对香港部分地区（迪士尼、西九龙）进行了3D建模，网页实现了区域转换、特效包括：下雨、下雪和烟花（仅在迪士尼）、路线和测量等功能。
